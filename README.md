# ✨ Projeto Frontend - React + Vite

Este é o projeto frontend de uma aplicação full stack, desenvolvido com **React** utilizando o **Vite** para um ambiente mais rápido e moderno.  
Atualmente, o frontend já está integrado, de maneira simples, ao backend e consome dados de usuários via API.

---

## 🚀 Tecnologias

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Axios](https://axios-http.com/) – para requisições HTTP
- [Node.js (API)](https://nodejs.org/) – backend conectado
